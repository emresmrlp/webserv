What is Webserv?

Webserv is a high-performance HTTP server developed in C++ 98. It is a systems programming project designed to replicate the core mechanics of modern web servers like Nginx or Apache. The engine handles network communications, parses HTTP protocols, and serves resources to multiple clients simultaneously using non-blocking I/O multiplexing.
